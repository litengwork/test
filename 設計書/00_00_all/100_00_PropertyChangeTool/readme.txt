1.はじめに

PocketDocMenuは、各種の開発用Excel資料を利用した「Oracle」、「SQLServer」、
「MySQL」、「PostgreSQL」向けの開発支援ツールです。
本ソフトウェアはExcelにアドインとして登録し使用します。
本ソフトウェアは株式会社パワーステーションが提供する技術ドキュメントの内、
特定フォーマットのExcelファイルに対してのみ有効です。

特定フォーマットは以下の通りです。
  ・テーブル定義書_テンプレート.xls
  ・表領域一覧_テンプレート.xls
  ・ユーザ一覧_テンプレート.xls
  ・シーケンス一覧_テンプレート.xls
  ・IFファイル定義書_テンプレート.xls
  ※上記ファイルのフォーマットを変更すると正しく動作しない可能性があります。


2.インストール・アンインストール

インストール・アンインストール作業は特に必要ありません。
Excelにアドオン登録する場合は、Excelを起動しメニューバーから[ツール-アドイン]を
選択します。参照ボタンを押下し、「PocketDocMenu.xla」を指定します。
メニューバーに[PocketDoc]が表示されれば登録完了です。
登録を解除する場合は、[PocketDocMenu]のチェックを外し、
「PocketDocMenu.xla」ファイルを削除するだけです。

※レジストリは使用していません。



3.使用方法

Excelを起動しメニューバーから[PocketDoc]を選択します。

　※各機能で記述されている『』は基本設計書のXLSドキュメントです。

  a)テーブル一覧更新
    『テーブル定義書』のテーブル一覧シートを更新します。

  b)DDL
    ｱ)Type
      作成対象のデータベースを選択します。
    ｲ)CREATE TABLESPACE
      『表領域一覧』を利用しSQLファイルを作成します。
    ｳ)CREATE USER
      『ユーザ一覧』を利用しSQLファイルを作成します。
    ｴ)CREATE TABLE
      『テーブル定義書』を利用しSQLファイルを作成します。
    ｵ)CREATE VIEW
      『テーブル定義書』を利用しSQLファイルを作成します。
    ｶ)CREATE INDEX
      『テーブル定義書』を利用しSQLファイルを作成します。
    ｷ)CREATE SEQUENCE
      『シーケンス一覧』を利用しSQLファイルを作成します。

  c)Entity
    ｱ)Type
      作成対象のエンティティを選択します。
    ｲ)VB用構造体作成
      『テーブル定義書』又は、『IF定義書』を利用し構造体文字列をクリップボードにコピー
      します。
    ｳ)C言語用構造体作成
      『テーブル定義書』又は、『IF定義書』を利用し構造体文字列をクリップボードにコピー
      します。
    ｴ)Java用クラス作成
      『テーブル定義書』又は、『IF定義書』を利用しクラス文字列をクリップボードにコピー
      します。

  c)Data
    ｱ)データシート作成
      『テーブル定義書』を利用し新規Bookにデータシートを作成します。
    ｲ)データファイル作成
      データシートよりCSVファイルを作成します。
    ｳ)データファイル読込
      CSVファイルを読み込みデータシートに展開します。
    ｴ)コントロールファイル作成
      『テーブル定義書』を利用しSQL*Loader用のコントロールファイルを作成します。
    ｵ)SQLファイル作成
      データシートよりINSERT用SQLファイルを作成します。
